## Problem

Calibration scores are written to `.clawdibrate/history/scores.jsonl` per repo, but there's no way to view them or track progress across repos without manually reading JSONL.

## Deliverable

1. **Central scoreboard** — `orchestrator.py` `save_score()` now also appends to `~/.clawdibrate/scoreboards/<repo-slug>.jsonl` (with `repo` field) after every calibration run.

2. **`/clawdbrt:scores` skill** — `src/skills/scores/SKILL.md`:
   - Default: show score history for current repo from `.clawdibrate/history/scores.jsonl`
   - `--repo /path`: specific repo
   - `--all`: all repos from `~/.clawdibrate/scoreboards/`, grouped by repo, sorted by most recent, with trend arrows (↑↓→)

## Implementation

- `clawdibrate/orchestrator.py`: `_central_scoreboard_path()` helper + `repo_root` param on `save_score()`
- `src/skills/scores/SKILL.md`: inline python3 script, no extra deps
