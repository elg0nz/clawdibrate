## Problem

A change that fixes one specific search pattern from one transcript is likely overfit. A change that addresses a failure category across multiple transcripts is generalization.

## Deliverable

Track and enforce failure diversity:

1. **Diversity score.** For each proposed AGENTS.md change, count how many distinct failure categories and how many distinct transcripts it addresses.
2. **Gate.** Changes addressing only 1 transcript and 1 failure category get flagged as potentially overfit. Require `--force` or a minimum diversity threshold to apply.
3. **Report.** Include diversity metrics in the calibration summary: "This change addresses N categories across M transcripts."
