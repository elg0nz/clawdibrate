## Problem

Agents hand-roll functionality that installed tools already provide (e.g., manual symlinking instead of using `npx skills add`). This wastes time and duplicates code.

## Expected Behavior

AGENTS.md should instruct agents to read docs of installed tools before implementing anything.

## Fix

Add to Tuning Rules or Known Gotchas:
- "Before implementing any capability, check if an installed tool already does it. Read its docs first."
