## Problem

Agents skip steps in the version workflow — starting work without a SPEC.md, committing without a README, or using checklists instead of kanban cards.

## Expected Behavior

AGENTS.md should document the mandatory version workflow.

## Fix

Add a "Version Workflow" section or expand Boundaries:
1. Create `docs/vX_Y_Z/SPEC.md` — define goals and scope
2. Create kanban cards from the spec
3. Copy `icebox.md` from prior version
4. Work the cards
5. Write `docs/vX_Y_Z/README.md`
6. Update `docs/CHANGELOG.md`, bump AGENTS.md version
7. Commit (README must exist first)
