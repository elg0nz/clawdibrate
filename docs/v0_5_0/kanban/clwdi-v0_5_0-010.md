## Problem

`clawdibrate-loop.py` defaulted to 20 iterations and stopped early at convergence (avg >= 0.95). User wants exactly 10 iterations (10 patch versions) by default, no early stop, and a `docs/vX_Y_Z/` folder with a CHANGELOG.md generated when the loop finishes.

## Fix Applied

- Changed default from 20 to 10 iterations
- Removed early convergence stop — loop always runs all iterations
- Added `--iterations`/`-n` flag to override
- After all iterations, creates `docs/vX_Y_Z/CHANGELOG.md` with per-iteration tuning summary and score progression
