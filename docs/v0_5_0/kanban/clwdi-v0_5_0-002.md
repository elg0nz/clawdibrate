## Problem

Agents use markdown checklists (`- [ ]`) or TaskCreate/TaskUpdate tools to track work. The project uses kanban cards in `docs/vX_Y_Z/kanban/` — checklists are flat and don't support the workflow.

## Expected Behavior

AGENTS.md should explicitly forbid checklists and task tools for work tracking, directing agents to kanban cards instead.

## Fix

Add to Boundaries:
- "🚫 Never: use markdown checklists or TaskCreate tools for work tracking — use `docs/vX_Y_Z/kanban/` cards"
