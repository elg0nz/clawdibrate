# Spec for this repo's logo

A bold, friendly crab mascot logo for an app called Clawdibrate. The crab is the hero — confident, slightly techy, with personality. One claw holds or displays a small gear (representing self-tuning), the other claw has subtle signal/wifi arc lines radiating from it (representing calibration and feedback loops). The crab has expressive dot eyes on stalks and a smooth rounded shell with subtle segmentation lines.
Color palette: deep burnt orange and warm rust (#E05C2A, #C44A1E) as the primary crab color, with off-white highlights and dark charcoal accents. Clean, flat illustration style — no gradients, no drop shadows.
Below the crab, the wordmark reads "Clawdibrate" in a modern geometric sans-serif, medium weight, with the tagline "self-tuning agents.md" in a small monospace font underneath, letter-spaced wide.
Overall vibe: Anthropic-adjacent, developer-tool aesthetic, playful but precise. Think Rust mascot meets dev tooling brand. Works on both light and dark backgrounds.
