# clwdi-v0_10_0-014: `--check-idempotent` convergence assertion

**Status:** todo
**Priority:** P2
**Size:** S
**Wave:** 1
**Depends on:** none

## Description
Add `--check-idempotent` flag that:
1. Runs calibrate once (normal)
2. Runs calibrate again on the same transcripts
3. Asserts edit_distance=0 on the second run (no changes)
4. Prints PASS/FAIL with details

This directly addresses Faros lightsaber check: "Running clawdibrate twice on the same transcript produces the same output."

## Acceptance
- `python -m clawdibrate --check-idempotent --transcript path.jsonl` runs and reports
- Exit code 0 = idempotent, 1 = divergence detected
