## Problem

Sections are skipped if their average score >= 0.8 in a single iteration. A section with 4 passing tasks and 1 critical failure gets skipped. The failing task is silently ignored.

## Fix

Skip only if the section scored >= 0.95 across 3+ consecutive calibration runs (use `converged_sections` tracking). In the new architecture, "iteration" means a full calibration pass over a set of transcripts, not a single task.
