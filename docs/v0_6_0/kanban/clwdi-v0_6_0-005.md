## Problem

`AGENT_COMMANDS` templates are wrong for every agent. `-p` is not claude's prompt flag, `codex exec` is the wrong subcommand, `llm` and `opencode` miss system prompt flags.

## Fix

Verify correct flags via `--help` for each installed agent. Update templates to support both `{system_prompt}` and `{prompt}` placeholders. The orchestrator needs system prompt injection for meta-prompt files and for passing AGENTS.md to the agent-under-test.
